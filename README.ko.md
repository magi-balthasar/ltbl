# LTBL — 빛이 있으라

> *"조건을 선언한다. 결과는 창발한다."*

**한국어 | [English](README.md)**

---

## 두 가지 탐구 질문

**Q1.** 의식은 언제, 어떤 조건에서 자체적으로 창발하는가?  
**Q2.** 창발한 의식은 어떻게 후대로 전달되고 발전하는가?

의식의 창발과 유전적 전달은 분리된 문제가 아니다.  
전달되지 못하는 의식은 진화적으로 의미가 없다.  
전달 메커니즘 자체가 의식의 발전 방향을 결정한다.

---

## 핵심 철학

- **몸이 먼저, 감각이 두 번째, 알고리즘은 마지막** — 지능은 물리적 피드백 루프에서 발생한다
- **대자연 = 가장 긴 시뮬레이션** — 생물 진화 순서대로 재현한다
- **보상 함수를 설계하지 않는다** — 압력과 조건만 설계한다
- **의식을 만들려 하지 않는다** — 의식이 나타날 조건을 만든다
- **전달되지 못하는 의식은 없는 것과 같다**

## 7단계 로드맵

| Phase | 모델 | 의식 레벨 | 유전 메커니즘 |
|-------|------|-----------|--------------|
| **1** | 단세포 (태초의 바다) | C0–C2 | 자기복제, 변이, 선택 |
| 2 | 분산 신경망 (히드라, 해파리) | C2–C3 | 정보 안정화 (DNA 등가물) |
| 3 | 절지동물 (CPG 운동) | C3–C4 | 재조합 (유성생식 등가물) |
| 4 | 어류 (척수-뇌간) | C4 | 구조적 복잡성 전달 |
| 5 | 4족 보행 | C4–C5 | 행동 패턴 전달 + 학습 전달 |
| 6 | 사회적 학습 | C5–C6 | 문화적 전달 (유전자 외) |
| 7 | 추상화 | C6 | 언어를 통한 정보 압축 전달 |

## Phase 1: 태초의 바다 (현재)

### 환경

- **유체**: 2D 화학 농도 그리드, 확산 방정식 기반
- **에너지원**: 열수공(hydrothermal vent) — 위치와 강도는 랜덤, 조석 효과로 맥동
- **위험**: 독소 패치 국소 분포
- **주기성**: 조석 주기, 열수공 맥동, 농도 확산

### 에이전트 (CellAgent)

```
[환경 화학 농도]
      │
   8개 센서 (가중치 진화 가능)
      │
   내부 상태 (에너지, 독소, 이력)
      │
   항상성 압력 → 행동 긴박도 결정
      │
   액추에이터 (이동 방향/속도)
      │
[소비 / 독소 흡수 / 에너지 소모]
```

외부 보상 함수 없음 — 내부 에너지 항상성 압력만이 행동을 유발한다.

### 유전 시스템

세 가지 복제 전략을 병렬 실험한다:

| 전략 | 방식 | 특징 |
|------|------|------|
| **무성생식** | 복사 + 랜덤 변이 | 기준선 |
| **유성생식** | 두 부모 재조합 + 변이 | 다양성 증가 |
| **라마르크** | 생애 경험을 게놈에 압축 전달 | 획득 형질 유전 |

최적 변이율은 실험적으로 발견한다 — 너무 정확하면 진화가 없고, 너무 부정확하면 정보가 소멸한다.

### 의식 레벨 측정

| 지표 | 설명 | 도입 Phase |
|------|------|-----------|
| **C0** | 내부 상태 변수의 수와 활성도 | 1-A |
| **C1** | 내부 상태 기반 행동의 비율 | 1-A |
| **C2** | 시간적 비교 깊이 (에너지 이력 분석) | 1-A |
| C3 | 예측 정확도 × 예측 시간 범위 | 1-C |
| C4 | 자기 모델 정확도 | 1-C+ |
| C5 | 타자 모델 정확도 | 1-D |
| C6 | 재귀적 자기 참조 발생 여부 | Phase 7 |

## 실행

### 설치

```bash
uv venv && source .venv/bin/activate
uv pip install -r requirements.txt
```

### 단일 섬 (디버그)

```bash
python main.py --single --steps 200 --report 20
```

### 병렬 실험 (8개 섬 × Ray)

```bash
python main.py --steps 500 --report 10 --intervene 50
```

```
┌── Step    10 ─────────────────────────────────────────────────┐
│ [0] asexual      μ=0.005 pop= 312 gen=  4.2/ 7  E= 2.31  C=1 ...
│ [3] sexual       μ=0.010 pop= 289 gen=  3.8/ 6  E= 2.18  C=1 ...
│ [5] lamarckian   μ=0.010 pop= 301 gen=  3.1/ 5  E= 2.44  C=1 ...
└───────────────────────────────────────────────────────────────┘
```

### 옵션

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| `--steps` | 200 | 매크로 스텝 수 |
| `--report` | 10 | 보고 주기 |
| `--intervene` | 50 | 신의 개입 주기 |
| `--db` | `logs/ltbl.db` | SQLite 저장 경로 |
| `--single` | off | 단일 섬 디버그 모드 |

관찰 데이터는 `logs/ltbl.db`에 누적된다. SQLite로 직접 쿼리 가능:

```sql
SELECT replication_mode, AVG(c1), MAX(max_generation)
FROM observations
GROUP BY replication_mode;
```

## 병렬 실험 설계

```
섬 0: asexual    μ=0.005  ─┐
섬 1: asexual    μ=0.020   │
섬 2: asexual    μ=0.080   ├─ 격리 → 다양성
섬 3: sexual     μ=0.010   │  주기적 Migration → 수렴
섬 4: sexual     μ=0.040   │
섬 5: lamarckian μ=0.010   │
섬 6: lamarckian μ=0.040   │
섬 7: high-stress μ=0.030 ─┘  (소형 세계, 열수공 1개)
```

**신의 개입 사이클 (N 스텝마다):**
1. 전체 의식 레벨 스캔
2. 소멸 위기 섬 → 자원 재배분
3. 최고 의식 레벨 섬 간 교차수분
4. 전 이력 SQLite 기록

## Phase 전환 조건

Phase 1 → Phase 2 전환 기준:
- 에너지 항상성 유지율 > 80%
- C1 (내부 주도 행동) > 0.5 안정적 달성
- 유전 전달로 의식 레벨 기준선이 세대를 거듭하며 상승
- Phase 2 신호 감지 (집단적 반응 패턴)
